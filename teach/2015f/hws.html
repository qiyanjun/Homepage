Course Project

  • Each project should be done in a team of 1 to 3 students.
  • Each student's class project is expected to span about 10 weeks.
  • Class project will be handled according to the following policies. -->

General timeline about the project:

  • Week10: Submission of project mid-phase report;
  • Week15: Submission of presentation slides;
  • Week16: Project presentation. Each presentation will be 10-minutes long with 2 minutes for questions.
  • Week16: Submission of final report. Each report is expected to be at least 9 page long. A latex template will be provided.
